How I work

Research depth, engineering discipline.

The value is not a model in isolation. It is a dependable path from a real constraint to an output a person can act on.

01 / FRAME

Start with the workflow

Define the decision, user, data boundary and acceptable trade-offs before choosing an architecture.

02 / BUILD

Train with reality in mind

Design datasets and evaluation around messy footage, edge cases, privacy and operational limits.

03 / DEPLOY

Own the last mile

Package inference for cloud GPUs, Docker or offline Windows delivery, depending on the constraint.

04 / LEARN

Close the loop

Iterate with editors, investigators, researchers and non-specialists until the output is genuinely useful.
